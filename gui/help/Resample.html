Overview of the S+Resample Package

Welcome to the S+Resample package!

There are three main ways to use the package: using modified versions of the graphical interface for certain specific applications such as comparing two means, using resampling menus (bootstrap, jackknife, influence, permutation tests) which support a wide variety of statistics, or using the Command line.

Supplements to Other Menus for Specific Applications

There are currently resampling supplements to six statistical analysis menus:
  • Summary Statistics: Statistics Data Summary Summary Statistics...
  • Correlation: Statistics Data Summary Correlations/Resample...
  • Proportions: Statistics Compare Samples Counts and Proportions Proportions Parameters/Resample...
  • One-sample t test: Statistics Compare Samples One Sample t Test
  • Two-sample t test: Statistics Compare Samples Two Samples t Test
  • Linear Regression: Statistics Regression Linear/Resample...

    Menus for Resampling, for use with General Statistics

    There are three menus for different kinds of resampling, which may be used with very general statistics:
  • Bootstrap: Statistics Resample Bootstrap...
  • Jackknife and Influence: Statistics Resample Bootstrap...
  • Permutation Tests: Statistics Resample Permutation Test...

    Command Line

    There are a large number of resampling functions available from the command line. An extensive overview with an annotated list and links is available at
    resampleOverview. A quick list is included here.

    High-level resampling functions include:


    bootstrap
    bootstrap2
    influence
    jackknife
    parametricBootstrap (parametric bootstrap)
    smoothedBootstrap (smoothed bootstrap)
    permutationTest
    permutationTest2 (permutation test for comparing two samples)
    permutationTestMeans (permutation test for the difference between means of two samples)
    crossValidation (cross-validation)
    bootstrapValidation (bootstrap validation, i.e. bootstrapping prediction error)
    parametricBootstrapTest (parametric bootstrap tests)

    Generic functions that operate on the objects created by the resampling functions include:
    print,
    plot,
    summary,
    qqnorm,

    Confidence intervals:


    limits.abc
    limits.bca
    limits.tilt
    limits.t (t intervals using resampling standard errors)
    bootstrapT (bootstrap t intervals)
    limits.percentile (this gives simple quantiles of a bootstrap distribution)

    Diagnostics:


    jackknifeAfterBootstrap,
    tiltAfterBootstrap,
    reweight.
