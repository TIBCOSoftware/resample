Sampler

Select the sampling method. The default, samp.bootstrap, does ordinary bootstrap sampling.

Some of the sampling methods have optional or required arguments; the latter are indicated in the drop-down list by a "?". One common optional argument is size, which allows you to specify that resamples should be a different size than the original number of observations.

Sampling with replacement

A number of functions do bootstrap sampling, with some variations:
samp.boot.bal is useful if your goal is to estimate bias.
samp.bootknife gives better bootstrap standard errors--the ordinary bootstrap estimate of the squared standard error (variance of a statistic) is biased downward by a factor of (n-1)/n, where n is the sample size. Another way to avoid this bias is to use samp.bootstrap and set size = n-1.
samp.finite is for bootstrapping from finite populations; set N to the population size.
samp.blockBootstrap samples consecutive blocks of observations, rather than individual observations. This is a quick-and-dirty way to resample time series.

Sampling without replacement

samp.permute does sampling without replacement. When size = n this gives permutations.
samp.combinations does all possible permutations or combinations.
