Subject

Specify a variable containing subject identification codes; the unique values of this variable indicate subjects. The subjects are then resampled, not individual observations - if a subject is selected for a bootstrap sample k times, then all observations for the subject are included k times.

If Data Set is a data frame, then this may be a variable in the data frame, or it may be a vector of the same length as the number of rows in Data Set.

This may also be used for cluster sampling.
