Statistic

Specify the statistic to be resampled. The statistic should normally return a numerical object, a vector or matrix. However, it may return NULL, when resampling for graphics rather than for numerical calculations.

This may be a function that accepts an lm (linear regression) object as the first argument, e.g.
coef pulls the coefficients out of an lm object.

Or it may be an expression that calculates the statistic, e.g. if you want only the second (slope) coefficient,
coef(fit)[2]

In the case of an expression, if you saved the lm object on the Model page, use that name here. For example I used "fit" just above. Otherwise use the name "data" in place of "fit".
