Difference between two samples

When resampling the difference between two samples, you may use either of the followng two fields:

Treatment

Specify the combined data set in the "Data" area, and a vector with two unique values as the Treatment field.

If the data set is a data frame, then Treatment may be the name of a variable in the data frame, e.g.
Sex.
Or it may be an expression that uses variable(s) in the data frame, e.g.
Dose < 15.

Data Set

Specify a data set containing the second sample. If the data set containing the first sample is a matrix or data frame, this must be too, with the same variables.
