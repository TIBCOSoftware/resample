Group

Specify a variable that indicates strata, for use in stratified resampling; the unique values of this variable indicate strata. If this is specified, sampling is done within each stratum (group). If Data Set is a data frame, then this may be a variable in the data frame, or it may be a vector of the same length as the number of rows in Data Set.

Multiple-sample problems

This may also be used for multiple-sample problems. Again, resampling is performed within each group.

However, the special case of resampling the difference between two samples is handled more easily using the Difference between two samples fields in the Model tab.
