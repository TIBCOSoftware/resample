Statistic to Estimate

In the Expression field, specify the statistic to be resampled. The statistic should normally return a numerical object, a vector or matrix. However, it may return NULL, when resampling for graphics rather than for numerical calculations.

This may be a function that accepts data as the first argument, e.g.
mean .

Or it may be an expression that calculates the statistic, e.g.
mean(x, trim = 0.1)

In the case of an expression, if the data set is a data frame, you may refer to variables by name. E.g if the data set is fuel.frame, then you may do:
mean(Fuel, trim = 0.1)
