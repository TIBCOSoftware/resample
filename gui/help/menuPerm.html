Permutation Test

In permutation testing one randomly reorders the data in a way consistent with a null hypothesis, in order to estimate the distribution of a statistic under the null hypothesis, and calculate a P-value.

To use permutation testing, you must supply the name of the data set under examination and an S-PLUS function or expression that calculates the statistic of interest.

The Permutation Test dialog performs permutation tests inferences for a specified statistic and data set. See the S-PLUS Resample Library User's Guide and Guide to Statistics for details.

To perform permutation tests,
choose Statistics Resample Permutation Test. The dialog shown below appears.

To test whether there is a significant relationship between two variables (e.g. correlation, or regression coefficient), only one of the variables should be permuted. This means that only one of the variables should be supplied as the Data Set.
