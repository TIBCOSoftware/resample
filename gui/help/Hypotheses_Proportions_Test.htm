Hypotheses

Proportions Variable Specify the variable that contains the hypothesized values for p, the probabilities of success. This is not required and it defaults to 0.5 for one group. If there is more than one group, the Null hypothesis is that the true probability of success p is the same for all groups.

Alternative Hypothesis Specify the alternative hypothesis.

. two.sided true parameter is not equal to p (default)

. less  true parameter is smaller than p

. greater  true parameter is greater than p.
