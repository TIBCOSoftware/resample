Group Variables

The variable names for the selected data set appear in the scrolling list. Select one or more variables to display the summary statistics in groups based on the values of that variable. To select more than one variable, click one then Ctrl-click any additional variables. If the variable selected is a factor, the data are grouped by each level of the factor. If the variable is numeric, then Maximum Unique Numeric Values and Number of Bins for Numeric Values are used.
