Linear Approximation

Specify a method for calculating a linear approximation to the statistic being resampled. This is used for determining the acceleration for BCa confidence intervals, for calculating tilting confidence intervals and diagnostics, and other uses.

The basic methods are:
jackknife: calculated by leaving out one observation at at time
influence: empirical influence function, calculated using a small epsilon,
regression: calculated by regression of resampled statistic values vs the frequencies of observations in the resamples, and
ace: like regression, except after the first regression, a nonlinear transformation is applied to the "y" (resampled statistic) values, and a second regression is run.

In addition,
choose: indicates to choose one of the above choices, based on the sample size, number of resamples, and other factors.
none: prevents a linear approximation from being calculated and saved (it is calculated if needed in any case).

Formula

Normally the regression and ace regress against observed frequencies. This regression has n-1 explanatory variables, where n is the number of observations in the data set. If n is large and the number of bootstrap samples is small the results may be inaccurate. You may specify the explanatory variables using a formula instead; e.g. if the statistic is variance, then a suitable formula would be ~x + x^2, where x is the variable being resampled.
