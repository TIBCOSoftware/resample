Maximum Unique Numeric Values

Specify a number of unique values for numeric columns. Numeric variables having more distinct values than this number are binned. The default value is 10.

In the Split Data by Group dialog, this field is related to the Group Column field. If the Group Column is numeric with no more unique values than the value specified, then a new data set is created for each unique value of the column. If there are more unique values than the number specified, then the data set is split into a number of new data sets or bins. The number of bins is determined by the value in Number of Bins for Numeric Values, and the bins are of equal width.
